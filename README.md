# TastySwifty 🍔🚀

TastySwifty is a full-stack food ordering web application built using the MERN stack.  
The platform allows users to browse food items, manage carts, place food orders, and track orders through a modern responsive interface.

The project contains:

- User Frontend Website
- Admin Dashboard
- Backend API Server
- MongoDB Database Integration

---

# 🌟 Features

## 👨‍🍳 User Frontend
- Browse food categories
- View food items with images
- Add/remove items from cart
- Place food orders
- User authentication
- Order tracking page
- Responsive UI

---

## 🛠️ Admin Panel
- Add new food items
- Upload food images
- Remove food items
- View all orders
- Manage delivery status

---

# ⚙️ Backend
- REST API using Express.js
- MongoDB database connection
- JWT Authentication
- Multer image upload system
- Stripe payment integration
- Order management system

---
